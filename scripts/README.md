# Scripts e Ferramentas de Desenvolvimento

Esta pasta contém scripts utilitários, arquivos de debug e ferramentas de desenvolvimento.

## Estrutura

- **debug-***.html/js**: Arquivos HTML/JS para debug de funcionalidades específicas
- **test-***.html/js**: Scripts de teste e validação
- **fix-***.ps1/js**: Scripts PowerShell/JS para correções e manutenção

## Uso

⚠️ **Atenção**: Estes scripts são para uso em desenvolvimento/teste. Não executar em produção sem revisão.

## Scripts Disponíveis

### Debug
- `debug-conversations.html` - Debug de conversas
- `debug-media.js` - Debug de mídia

### Testes
- `test-photos.html` - Teste de fotos
- `test-evolution-config.js` - Teste de configuração Evolution API
- `TEST_IMAGE_ACCESS.html` - Teste de acesso a imagens

### Fixes
- `fix-incomplete-policy-drops.ps1` - Corrige políticas incompletas
- `fix-migrations.ps1` - Corrige migrations
- `fix-policies.ps1` - Corrige políticas RLS
- `fix-storage-cors.js` - Corrige CORS do storage
- `fix-triggers.ps1` - Corrige triggers
